# README Assignment 1
## Exercise 1
The script takes the price of food and drink as well as tip percentage as inputs, and returns the prices for food drink, and the total bill amount with and without tip. Use this script to calculate the bill based on the purchases. 
## Exercise 2
The script takes a list of four assignment scores as input, and returns the average score as well as whether the student passed or failed, depending on if the average score is above or below 50. Use this script to calculate grades of students based on test scores. 
## Exercise 3
The script takes a list of three assignment scores as input, and returns the individual scores, the average scores and the grade of the student based on the average score. It also raises a warning if any of the test scores is below 40. Use this script to calculate grades of students based on test scores.
